## 💡 About the Project
This simulation aims to visualize why planets behave differently under Albert Einstein's General Relativity compared to Isaac Newton's Universal Gravitation. 

### Key Visualizations:
*   **Newton Panel:** Shows stable, unchanging elliptical orbits where space is a fixed stage.
*   **Einstein Panel:** Visualizes **Space-Time Warping**. Gravity is shown as the curvature of a dynamic grid, leading to **Orbital Precession** (the famous "rosette" pattern).

## ✨ Features
*   **Bilingual Support (TR/EN):** Instantly toggle between Turkish and English.
*   **Dynamic Space-Time Grid:** A relativistic grid that reacts to the mass of the star and the planets.
*   **Multi-Planet Support:** Simulate up to 5 planets simultaneously with unique orbital dynamics.
*   **Path Prediction:** Real-time future path tracing to visualize precession before it happens.
*   **Performance Optimized:** Optimized rendering logic maintaining 60 FPS even with complex grid warping.

## 🛠️ Controls
*   **Star Mass:** Adjust the gravity strength of the central star.
*   **Simulation Speed:** Speed up time to see centuries of orbital shift in seconds.
*   **Advanced Settings:** Fine-tune initial velocity, relativity multiplier, and planet count.

## 👨‍🔬 Technical Details
*   **Core:** Vanilla JavaScript (ES6+)
*   **Rendering:** HTML5 Canvas API
*   **Physics:** 4th-order integration approximation with relativistic correction factors.
